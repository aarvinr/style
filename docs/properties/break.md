# Break

The `break` property can be used as an alias for `break-before`, `break-after`, and `break-inside`.

<!-- tabs:start -->

### **Style**

```css
:root {
  break: avoid right revert;
}
```

### **CSS**

```css
:root {
  break-before: avoid
  break-after: right;
  break-inside: revert;
}
```

<!-- tabs:end -->
