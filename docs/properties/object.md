# Object

The `object` property can be used as an alias for `object-fit` and `object-position`.

<!-- tabs:start -->

### **Style**

```css
:root {
  object: cover;
  object: cover top;
  object: contain 25% 75%;
}
```

### **CSS**

```css
:root {
  object-fit: cover;

  object-fit: cover;
  object-position: top;

  object-fit: contain;
  object-position: 25% 75%;
}
```

<!-- tabs:end -->
